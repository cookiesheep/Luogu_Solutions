# B2025 输出菱形字符
## 题目描述
用 * 构造一个对角线长 5
5 个字符，倾斜放置的菱形。

输入格式
没有输入要求。

输出格式
如样例所示。用 * 构成的菱形。
```
输入输出样例
输入 #1复制
输出 #1复制
  *
 ***
*****
 ***
  *
```
